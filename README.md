# [Holmofy Blog](https://blog.hufeifei.cn)

base on [hexo](https://hexo.io/zh-cn/)

## Install

**1. clone repo**
```
git clone <url>
cd blog.hufeifei.cn
```
**2. install theme**

[holmofy/paper](https://github.com/holmofy/hexo-theme-paper)

**3. compile and deploy**
```
npm install
hexo clean && hexo g && hexo d
```
